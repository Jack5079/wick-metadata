What does this even do? This tool lets you see metadata for Wick Editor projects.
Who made this? Hi! I'm Jack. I make games sometimes.
What projects can I scan? Projects that fit these critera:
  • Made in 1.0 or newer
  • File type is a .wick
What metadata is shown?
  • The project's name
  • The resolution
  • A framerate
  • The browser used to save it
  • The OS that the browser was ran on
  • The background color
