FAQ
What data you can get from this?

In order of appearance from top to bottom:

  1. The background color (Border color)
  2. The project's name
  3. The resolution
  4. A framerate
  5. The browser and OS used to save the project
  6. The .wick file
What files are supported?

New Wick Editor projects.

How do you get the OS and browser info?

To find it yourself, rename your .wick to a .zip, open project.json, and search for "platform"

This data was added to the project.json format in commit 34418df[1].

Apparently, this was added for "debugging wick projects."[2]
